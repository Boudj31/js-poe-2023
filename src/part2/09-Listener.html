Les Event Listener

Les événement en JS permettent de détecter et réagir aux interactions de l'utilisateur.

Ils sont asyncrones

Event de souris

click: se déclenche lorsque un élément est cliqué mouseover: se déclenche lorsque le curseur de la souris survole un élément mouseout : se déclenche lorsque le curseur de la souris quitte un élément

Event de clavier

keydown : lorsque qu'on appuie sur une touche keyup : lorsque qu'on relache la touche keypress : lorsque qu'on maintient la touche enfoncé

Event de fenêtre

scroll : lorsque l'on fait défiler la page resize : lorsque la fenêtre est redimensionné

Event de chargement

DOMContentLoaded : se declenche lorsque le DOM est entièrement construit load : lorsque la page et toutes ses ressources sont entièrement chargées
